다음 WIP 작업을 취소 처리해주세요:

[WorkID와 사유]: $ARGUMENTS

**작업 절차**:
1. WORK_IN_PROGRESS.md에서 해당 WorkID 상태 확인
2. 취소 사유 기록
3. 취소 상태로 변경:
   - WORK_IN_PROGRESS.md 활성 작업 테이블에서 해당 행 제거
   - WORK_IN_PROGRESS.md 작업 상세 섹션에서 해당 WorkID 블록 제거
   - WORK_HISTORY.md 취소된 작업 테이블에 행 추가 (취소일, 사유 포함)
4. `.wips/active/` 내 해당 WIP 파일을 `.wips/archive/`로 이동
5. 관련 리소스 정리

**요구사항**:
- 취소 사유 명확히 기록
