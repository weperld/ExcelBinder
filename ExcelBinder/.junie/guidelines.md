# Junie 가이드라인

Junie는 ExcelBinder 프로젝트에서 작업할 때 **AGENTS.md**의 내용을 반드시 준수해야 합니다.

## 필독 문서

1. **AGENTS.md** - 전체 작업 가이드 및 지시 템플릿
2. **PROJECT_SUMMARY.md** - 프로젝트 30초 요약
3. **WORKFLOW_PLANNING.md** - 기획서 처리 시스템
4. **WORK_IN_PROGRESS.md** - 현재 진행 중 작업 확인

## 원칙

- 모든 작업은 AGENTS.md의 지시를 따릅니다
- WORK_IN_PROGRESS.md를 자동으로 업데이트합니다
- 추가적인 규칙은 AGENTS.md를 참조하세요
